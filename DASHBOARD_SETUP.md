# 🎯 إعدادات لوحة التحكم - aizo.dz

## ✅ متطلبات قبل البدء

1. **Node.js و npm** - مثبتة على الكمبيوتر
   - تحقق: افتح PowerShell وكتب `node --version`
   - إذا لم تكن مثبتة: https://nodejs.org

2. **المكتبات المطلوبة** - مثبتة
   ```powershell
   cd c:\Users\pc\Desktop\aizo-dz
   npm install
   ```

## 🚀 بدء تشغيل لوحة التحكم

### الخطوة 1: تشغيل السيرفر
```powershell
cd c:\Users\pc\Desktop\aizo-dz
npm start
```

ستراه رسالة:
```
Server is running locally: http://localhost:3000
```

### الخطوة 2: الدخول إلى لوحة التحكم
افتح المتصفح واذهب إلى:
```
http://localhost:3000/admin.html
```

### الخطوة 3: تسجيل الدخول
- **كلمة السر الافتراضية:** `admin123`

## ⚠️ عند مواجهة مشاكل

### المشكلة: لا ينقلني إلى الداشبورد بعد إدخال كلمة السر

**الحل:**
1. اضغط `F12` لفتح أدوات المطور (Developer Tools)
2. اذهب إلى تبويب "Console"
3. ابحث عن الرسائل الحمراء (الأخطاء)
4. تأكد من أن السيرفر يعمل (ستظهر رسالة حمراء إذا لم يكن متصلاً)

### المشكلة: رسالة تحذيرية حمراء "الخادم غير متصل"

**الحل:**
- تأكد من تشغيل السيرفر بأمر `npm start`
- تحقق من أنك فتحت اللوحة على `http://localhost:3000` وليس ملف محلي

### المشكلة: npm لم يتم تثبيته

**الحل:**
1. حمل Node.js من https://nodejs.org
2. اختر الإصدار LTS (Long Term Support)
3. اتبع خطوات التثبيت
4. أعد تشغيل PowerShell والمحاولة مجدداً

## 💡 نصائح مفيدة

- **حفظ كلمة السر:** يتم حفظها في `sessionStorage`، لذا إذا أغلقت التبويب ستحتاج إلى إعادة تسجيل الدخول
- **بيانات المتجر:** يتم حفظ الطلبات والمنتجات في مجلد `db/`
- **الصور:** استخدم روابط مباشرة (URLs) للصور، مثل من Pinterest أو Imgur
