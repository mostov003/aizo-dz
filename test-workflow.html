🔄 اختبار نظام Workflow (Order Status Workflow Test)

الخطوة 1: قبول الطلب (Accept Order)

الحالة الحالية: قيد الانتظار (Pending)

الخيارات المتاحة: قبول / رفض

الخطوة 2: تأكيد الطلب (Confirm Order)

الحالة بعد القبول: في الانتظار (en attente)

الخيارات المتاحة: تأكيد / إلغاء

الخطوة 3: الشحن (Shipping)

الحالة بعد التأكيد: مؤكد (confirmé)

الخيارات المتاحة: تم الشحن / لم يتم الشحن

📊 ملخص Workflow

الحالة 1 - Pending: يتم عرض أزرار (قبول / رفض)
الحالة 2 - en attente: يتم عرض أزرار (تأكيد / إلغاء)
الحالة 3 - confirmé: يتم عرض أزرار (شحن / عدم شحن)
الحالات النهائية: livré / retourné / annulé - لا توجد خيارات إضافية
